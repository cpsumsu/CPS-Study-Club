# 以下C++函数的功能是

```c++
int func(char* x)
{
	char* p = x;
	while (*p++);
	return (p - x - 1);
}
```

1. 将字符串x复制到字符串p中
2. 求字符串的长度
3. 将字符串x反向
4. 将字符串x连接到字符串p后面

# 答案

<details>

分析该C++函数的功能:
1. p是一个指向x的指针，没有复制字符串x
2. 使用while循环遍历字符串，直到遇到空终止符'\0'停止
3. 返回p和x之间的距离-1，即字符串x的长度
4. 没有反向或者连接字符串
所以该函数的功能是 **求传入字符串x的长度** 。

```c++
#include <iostream>
using namespace std;

int func(char* x)
{
	char* p = x;
	while (*p++);
	return (p - x - 1);
}

int main()
{
	char x[11] = "HelloWorld";
	cout << func(x) << endl; // cout 10
	return 0;
}
```

</details>
