# 以下方法的返回值是多少

```c++
int get_num()
{
    bool a = true;
    bool b = false;
    int x = 8;
    int y = x / 2;
    return (x | y) + (a + b);
}
```

# 答案

<details>

分析代码:
- bool a = true; // a 的值为 1 
- bool b = false; // b 的值为 0
- int x = 8;
- int y = x / 2; // y 的值为 4
- return (x | y) + (a + b);
- (x | y) 的值为 x 和 y 按位或运算的结果，8 和 4 的二进制表示分别为:
  - 8: 0000 1000
  - 4: 0000 0100
- 按位或运算的结果为:0000 1100，十进制值为 12
- (a + b) 的值为 1 + 0 = 1
- 所以返回值是 (x | y) + (a + b) = 12 + 1 = 13
所以该方法的返回值是 13。

```c++
#include <iostream>
using namespace std;

int get_num()
{
    bool a = true;
    bool b = false;
    int x = 8;
    int y = x / 2;
    return (x | y) + (a + b);
}

int main()
{
    cout << get_num() << endl;
    // cout: 13
	return 0;
}
```
